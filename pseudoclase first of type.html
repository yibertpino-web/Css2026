🎓 Pseudoclases utilizadas

  • :root → Variables globales
  • :first-child → Primera casilla
  • :last-child → Última casilla
  • :nth-child(even) → Casillas pares
  • :nth-child(odd) → Casillas impares
  • :first-of-type → Primer peón
  • :last-of-type → Último peón
  • :only-child → Rey animado
  • :empty → Casillas vacías
  • :hover → Efecto al pasar el mouse
  • :active → Al hacer clic
  • :focus → Al seleccionar
  • :target → Casilla seleccionada
  • :not() → Piezas ocupadas
