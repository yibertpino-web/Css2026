Ambiente de Formación - Desarrollo Web

  • Elemento 1 :first-child
  • Elemento 2 :nth-child(2)
  • Elemento 3 Normal
  • Elemento 4 :nth-child(even)
  • Elemento 5 Normal
  • Elemento 6 :nth-last-child(2)
  • Elemento 7 :last-child

📚 Explicación

✅ :root define variables globales CSS.

✅ :first-child selecciona el primer elemento hijo.

✅ :nth-child(2) selecciona el segundo elemento.

✅ :nth-child(even) selecciona todos los elementos pares.

✅ :nth-last-child(2) selecciona el segundo elemento contando desde el final.

✅ :last-child selecciona el último elemento hijo.

:root{
   --verde:#39A900;
}

li:first-child{
   border-right:10px solid green;
}

li:nth-child(2){
   background:yellow;
}

li:nth-child(even){
   border-left:10px solid blue;
}

li:nth-last-child(2){
   background:red;
}

li:last-child{
   border-right:10px solid purple;
}
