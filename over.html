Overflow con Imágenes y Texto

overflow: visible → El contenido puede salirse del contenedor. Este texto es largo para probar el desbordamiento.
overflow: hidden → El contenido que no cabe se corta y no se ve. Texto adicional que puede quedar oculto.
overflow: scroll → Siempre aparecen barras de desplazamiento. Este texto es mucho más largo para forzar el scroll dentro del contenedor.
overflow: auto → Solo aparece scroll si es necesario. Si el contenido es pequeño, no aparece nada extra.
